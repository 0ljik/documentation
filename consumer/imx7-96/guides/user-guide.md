# i.MX7 96Board - Meerkat<sup>速</sup> User Guide

**Registering Meerkat<sup>速</sup>**

- Please register your i.MX7 96Boards - Meerkat<sup>速</sup> [here](http://www.novtech.com/registration)
- Provide the serial # from the back side of Meerkat<sup>速</sup> board.
- Provide the requested information and select **Meerkat** from the version drop down.
